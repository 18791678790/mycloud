# 文件清单

```
- item/								        项目根目录
  - public/							      存放公共文件的文件夹
  - .eslintrc.js					    ESLint 代码检测配置文件
  - .gitignore						    Git 忽略文件列表
  - jsconfig.json					    JavaScript 开发所需的配置文件
  - package.json					    Node.js 项目配置文件
  - package-lock.json				  记录依赖项的具体版本
  - postcss.config.js				  PostCSS 的配置文件
  - README.md						      项目的说明文档
  - tailwind.config.js 				Tailwind CSS 的配置文件
```
